# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive UI component library with 40+ components
- Custom React hooks for common patterns
- Utility functions for formatting and validation
- Icon components for common use cases
- Page section components for rapid development
- Full documentation for hooks and utilities
- Smart contract deployment guides
- WalletConnect integration via Reown AppKit
- NFT minting functionality
- Multi-chain support (Base, Ethereum, Arbitrum)

### Changed
- Updated Next.js to version 16
- Migrated to Tailwind CSS v4
- Enhanced README with comprehensive documentation

### Fixed
- Build configuration for Web3 dependencies
- Webpack fallbacks for Node.js modules
- TypeScript configuration for Next.js

## [0.1.0] - 2025-12-03

### Added
- Initial project setup
- Basic WalletConnect integration
- NFT minting interface
- Smart contract templates
- Deployment documentation
