# Base WalletConnect App

This project is a decentralized application (dApp) built on the **Base** network using **Reown AppKit** (formerly WalletConnect Web3Modal).

## Objective
To build a high-quality dApp that integrates WalletConnect technologies and contributes to the Base ecosystem, qualifying for the WalletConnect weekly rewards program.

## Tech Stack
- **Network**: Base
- **SDK**: Reown AppKit / WalletConnect
- **Framework**: Next.js (planned)
